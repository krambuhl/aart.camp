`Stack` (`components/shared/Stack/index.tsx`) defaults to `alignment="center"`. Cross-axis alignment depends on direction: in a vertical Stack, `center` aligns children to the *horizontal* center of the container. When a Stack uses responsive `direction={{ xs: 'vertical', sm: 'horizontal' }}` and contains text, the default alignment will horizontally center text on `xs` while having no visible effect on `sm`. This often goes unnoticed because horizontal viewports look correct.

When a `Stack` contains text and direction is responsive, set `alignment` explicitly. Common pattern for a row that becomes a vertical stack on mobile: `alignment={{ xs: 'start', sm: 'baseline' }}` — `start` left-aligns text when stacked, `baseline` keeps title and metadata on the same baseline when in a row.
